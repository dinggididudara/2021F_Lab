 HTML: HyperText Markup Language
Client-server model of communications: server is listening for queries from client.
Name of different types of servers: Data server, Application server, Authentication server, Email server, Web server, Media server.
Web server: responds to HTTP requests.
Data Centers: special facilities that server farms is housed in.
Why do we need URL? naming mechanism to allow clients to request particular resources from the server
Example of URL and specify all components: https(Protocol)://www.facebook.com(Domain)/marketplace/?ref=app_tab(Query String)

Difference between static and dynamic web site
1. Static web site: consists only of HTML pages
2. Dynamic web site: use programs running on web servers to generate content dynamically

Why do we need DNS? to discover numeric IP address, DNS Address Resolution
Examples of gTLD: Generic Top Level Domain, .com .net .org
Examples of ccTLD: Country Code Top Level Domain, .ca .kr
How can you pass a user's information from the client to the web server? Request-Response Loop
Function of HTTP? Server waits request and responds with reponse code.
How a single web pages is displayed on a web browser? page's elements are downloaded from web servers. The browser transforms elements into visual representation.
Examples of HTTP request methods: GET,POST,HEAD,PUT,DELETE,CONNECT,TRACE,OPTIONS...
